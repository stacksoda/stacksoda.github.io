JSX的优点

1.声明式创建界面使得组件的概念更直观

2.代码动态创建界面使得页面的组合更为灵活

3.无需学习新的模板语言

约定

自定义组件必须以大写字母开头

  1. React 认为小写的tag是原生DOM节点，如div
  2. 大写字母开头为自定义组件
  3. JSX 标记可以直接使用属性语法， 例如<menu.item />
